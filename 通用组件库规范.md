# 通用组件库规范

## 功能

* 常用组件覆盖
* 自定义主题

## 开发规范

* 禁用 `css modules` 功能


## 目录规范

```
├── src
│   ├── components
│   │   └── button
│   │    |	└── index.tsx
│   │    |	└── index.{less | scss}
│   │    |	└── index.mdx
```

* 一个组件一个文件夹，文件名小写，多个单词用`-`隔开
* 样式采用 `less` 或 `scss`，具体看基于的原始组件库封装
* 每个组件搭配一个 `mdx` 文件，便于 `docz` 展示
